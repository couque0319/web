
# 웹사이트 기획하기 

 (만든 이유 -> 도움받는 대상 -> 효과)
## 만든 이유
 만든 이유 : 전공 공부를 할때 부족한 부분이나 헷갈리는 개념이 있을 때 다른 학우들의 학습 노트를 보며 공부에 도움이 되었으면 하는 마음
 도움받는 대상 : 학부생, 관련 분야를 처음 공부하는 사람, 복수전공자
 효과 : 여러 학생들의 생각과 공부 방법을 통해 자신의 학습 노트를 만드는 노하우와 학습법을 만들어낼수 있고 관련 전공에 대해서 모르는 부분을 학습 할수 있
